Project 4

In Project 4, this code creates an interactive application that utilizes the p5.js framework to build several scenes, including a startup screen, a main scene, and a help screen. In these scenes, the user can interact with the program's hedgehog character, whose behavior changes dynamically based on the user's mouse actions. The main scene is the core part of the application, in which the hedgehog character interacts actively. The background of the scene is powder blue to give a fresh, serene feel, and the rectangular area at the bottom uses a dark blue color to create visual contrast. The hedgehog's animation and behavior changes based on the user's mouse interaction. When hovering over the hedgehog character, the hedgehog's animation switches to show a “standing” posture, and the hedgehog's position moves slightly randomly, adding a cute dynamic effect. The hedgehog's sound effects are also an important part of the interaction. The sound effect plays when the mouse is pressed, or pauses if it is already playing. This design allows users to not only see the hedgehog's movements visually, but also through sound to enhance the interactive experience. In addition to the hedgehog interaction, the main scene also includes a help button that allows the user to jump to the help screen when clicked. Through button color changes and animation effects, the interface responds to the user's actions and enhances the sense of interaction.
