Project 1

Press keys “1” to “4”: draw sky yellow, purple, orange and black.
Press key “5”: draws a pink gradient.
Press key “6”: draw a blue gradient.
Press key “7”: drawsa green grandient.
Press key “8”: Draws a randomly colored star.
Press key “9”: enable the eraser, hold down the mouse to erase the current drawing.
Press key “0”: clear the canvas and revert to a white background.
Press key “s”: save the current canvas image as a PNG file.
